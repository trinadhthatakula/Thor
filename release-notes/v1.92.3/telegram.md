🚀 **Thor v1.92.3 is here!**

A lean & reliable release — smaller download, a fixed suspend, and a hardened open-source root engine.

**What's New:**

• 📦 **~48% smaller** direct-download APK (6.23 MB → 3.24 MB) — compressed dex, obfuscation & subset fonts.

• 🛠️ **Suspend fix**: suspending/unsuspending apps works on release builds again.

• ⚡ **Hang-proof root**: root detection can no longer freeze the app, and shell commands now report their real exit codes.

• 🔓 **Odin goes open source**: our in-house libsu fork is now a published Apache-2.0 library — `com.trinadhthatakula:odin`.

• 🧹 **Polish**: lint + Qodana cleanup and losslessly compacted vector icons.
