⚡️ **Thor v1.81.6 is out!**

A major update bringing advanced Freezer app filtering, direct APK/APKs sharing, root cache clearing, and critical filter fixes!

**Key Highlights:**
❄️ **Freezer App Filtering**: Partition user and system apps in the Freezer to prevent freezing critical components.
📥 **Import Disabled Apps**: Automatically scan and import disabled apps not tracked in the Freezer (hides settings button if none are left).
📦 **Direct APK/APKs Sharing**: Share actual APK files or zipped APKS packages (with installer-compliant manifest) instead of store links.
🧹 **Clear Cache (Root)**: Easily clear cache files for user/system apps directly from the dashboard.
🛠️ **Bug Fixes**: Fixed activity state filters in the AppList screen and upgraded internal dependencies.
