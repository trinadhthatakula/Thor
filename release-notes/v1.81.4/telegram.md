⚡️ **Thor v1.81.3 is out!**

A major visual and architectural update bringing shared element transitions, expressive motion, and optimized memory management!

🎨 **Shared Elements**: Seamless transitions for app icons and names when opening details.

❄️ **Freezer Screen Fix**: The FAB now shows immediately without waiting for privilege load.
⚙️ **Build**: Upgraded libraries and cleaned up redundant compiler flags.
