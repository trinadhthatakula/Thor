⚡️ **Thor v1.81.9 is out!**

A brand new update introducing interface personalization and gateway stability! Control your app navigation style with the Detailed View toggle, redirect easily using the Details button in the app info dialog, optimize performance across all devices with 3-level Animation Intensity control, and enjoy safe, deadlock-free Shizuku/Dhizuku/Root operations!

**Key Highlights:**
📖 **Detailed View Settings**: Choose between opening `AppInfoDialog` or navigating directly to the dedicated `AppDetails` screen when clicking an app.
🔍 **Dialog Redirection**: Added a "Details" option to `AppInfoDialog` (available on App List and Freezer screens).
🎬 **Animation Intensity Control**: Configure animations at three levels:
• **Low**: Disables transitions and shared elements (instant snap navigation).
• **Medium**: Retains screen transitions but disables shared elements.
• **High**: Enables all animations and shared element transitions for a premium, expressive feel.
🔄 **Reinstall in Shizuku/Dhizuku**: Batch reinstall ("Fix Store") is now enabled for Shizuku and Dhizuku privilege modes in the multi-select toolbar.
🔒 **Gateway Safety Refinements**: Thread-safety offloads, shell escaping sanitization, regex package validation, concurrent process stream consumption to prevent deadlocks, and fully localized suspended app dialog prompts.
