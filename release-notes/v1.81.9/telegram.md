⚡️ **Thor v1.81.9 is out!**

A brand new update introducing user interface personalization! Control your app navigation style with the new Detailed View toggle, redirect easily using the new Details option in the app info dialog, and optimize performance across all devices using the new 3-level Animation Intensity control!

**Key Highlights:**
📖 **Detailed View Settings**: Choose between opening the compact `AppInfoDialog` or redirecting directly to the dedicated `AppDetails` screen when clicking an app icon.
🔍 **Dialog Redirection**: Added a new "Details" action item to the `AppInfoDialog` (available on both App List and Freezer screens) to quickly redirect to the detail screen.
🎬 **Animation Intensity Control**: Configure animations at three levels:
• **Low**: Disables standard screen transitions and shared transitions (instant snap navigation).
• **Medium**: Retains standard navigation transitions but disables shared elements.
• **High**: Enables all animations and shared element transitions for a premium, expressive feel.
