⚡ **Thor v1.92.1 is here!** ⚡

A massive stability and feature release packed with next-gen engine upgrades and highly-requested capabilities!

**What's New:**

• 🔄 **Auto Reinstall Apps**: Easily reinstall uninstalled or frozen apps with custom setup overrides.

• 🥶 **Odin Core Engine**: Upgraded our root process engine (`:suCore`) to a custom, pure-Kotlin powerhouse. Runs entirely in-memory with lightning-fast 2ms actions and zero legacy files.

• 🔒 **Security Hardened**: Armed with military-grade Binder UID verification. Only trusted processes can call privileged APIs.

• 🚀 **ANR-Free Threading**: Moved all IPC and database actions off the main thread—no more interface stutters or lags.

• 🎨 **UI Elegance Pass**: Brand new styling for freezer prompt action buttons and smooth marquee scrolling on tap for long settings texts.

• 🩹 **Custom ROM Fixes**: Resolved standard platform crashes on Android 8.0–12L and optimized API offset caching for seamless platform bypass.
