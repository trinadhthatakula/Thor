<p><b>Thor App Manager</b> is a modern, open-source Android app manager and installer, built for power users and enthusiasts who demand full control, privacy, and speed — without the bloat.</p>

<p>Thor is built 100% in Kotlin with Material 3 design and Jetpack Compose. It delivers a complete, privileged app management suite at around 2 MB Play Store download size — with zero ads, zero trackers, and zero internet required.</p>

<p><b>Privilege Modes — Root, Shizuku &amp; Dhizuku:</b></p>
<ul>
  <li>Works with Root (su), Shizuku, or Dhizuku Device Owner</li>
  <li>Manually select your preferred privilege engine (Root / Shizuku / Dhizuku)</li>
  <li>Automatic fallback chain if the preferred mode is unavailable</li>
</ul>

<p><b>App Management:</b></p>
<ul>
  <li>Install, uninstall, freeze (disable/enable), and reinstall apps</li>
  <li>Suspend and unsuspend apps with a custom Thor-branded system dialog</li>
  <li>Restrict app background activity</li>
  <li>Force-stop apps and clear app data or cache</li>
  <li>Uninstall and freeze/unfreeze system apps</li>
  <li>Fix Store — reassign installer record to Google Play in any privilege mode</li>
  <li>Share APK files directly</li>
  <li>Launch app activities</li>
</ul>

<p><b>Batch Operations:</b></p>
<ul>
  <li>Batch freeze / unfreeze, reinstall, uninstall, kill, suspend, and clear data</li>
  <li>Real-time terminal logger dialog during batch actions</li>
</ul>

<p><b>App List &amp; Insights:</b></p>
<ul>
  <li>High-performance app list with Room DB metadata caching</li>
  <li>Live search by app name or package name</li>
  <li>Sort and filter apps, including by installation source</li>
  <li>Split APK indicator for multi-APK apps</li>
  <li>App state badge: frozen, suspended, and hidden states</li>
  <li>Installer source, version code, target SDK, and debuggable flag</li>
</ul>

<p><b>Customization &amp; Accessibility:</b></p>
<ul>
  <li>Dark, Light, and System themes + true AMOLED black mode</li>
  <li>Material You dynamic color support (Android 12+)</li>
  <li>Asgardian static color scheme (default)</li>
  <li>Biometric / fingerprint lock for app access</li>
  <li>Multi-language support: English, Spanish, French, Arabic, Chinese</li>
  <li>In-app language switcher — no system settings required</li>
</ul>

<p><b>Privacy &amp; Performance:</b></p>
<ul>
  <li>100% offline — no internet permission</li>
  <li>No analytics, no crash reporters, no trackers</li>
  <li>Fully reproducible, copylefted libre software (GPLv3.0)</li>
  <li>~2 MB Play Store download; &lt;4 MB installed APK</li>
</ul>

<p><b>Upcoming Features:</b></p>
<ul>
  <li>App data backup and restoration</li>
  <li>Advanced packages.xml editing</li>
  <li>Batch APK install</li>
  <li>And more</li>
</ul>

<p><b>Technical Highlights:</b></p>
<ul>
  <li>100% Kotlin codebase with Clean Architecture + MVVM</li>
  <li>Custom in-house hidden API bypass module (no third-party dependency)</li>
  <li>Uses a modularized, fully Kotlin-converted fork of <a href="https://github.com/topjohnwu/libsu">libsu</a> as <i>suCore</i> for root operations</li>
</ul>
